### 爬取链家上海租房信息

### 文件
lianjia_spider.py 利用requests+正则获取链家上海租房信息。

lianjia_bs4_v1.py 利用requests+BeautifulSoup获取链家上海租房信息，并将结果保存到csv文件。

lianjia_bs4_v2.py 利用requests+BeautifulSoup获取链家上海租房信息，并将结果保存到mongodb中。
